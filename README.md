День 2 — Парсинг HTML и извлечение данных

🎯 Цель дня
Добавить функционал парсинга HTML-контента и извлечения структурированных данных.

📋 Требования

1. Установка библиотек для парсинга
```bash
pip install beautifulsoup4 lxml
```

2. Класс HTMLParser
Создать класс HTMLParser с методами:
- async def parse_html(self, html: str, url: str) -> dict — парсинг HTML
- extract_links(self, soup: BeautifulSoup, base_url: str) -> list[str] — извлечение всех ссылок
- extract_text(self, soup: BeautifulSoup, selector: str = None) -> str — извлечение текста
- extract_metadata(self, soup: BeautifulSoup) -> dict — извлечение мета-тегов (title, description, keywords)

3. Интеграция с AsyncCrawler
Расширить AsyncCrawler:
- Добавить метод async def fetch_and_parse(self, url: str) -> dict
- Метод должен возвращать словарь с полями:
  - url — исходный URL
  - title — заголовок страницы
  - text — основной текст
  - links — список ссылок (абсолютные URL)
  - metadata — метаданные

4. Обработка относительных ссылок
- 🔄 Конвертировать относительные ссылки в абсолютные
- 🔍 Фильтровать внешние ссылки (опционально)
- ✅ Валидировать URL перед добавлением

5. Извлечение специфичных данных
Реализовать методы для извлечения:
- 🖼️ Всех изображений с атрибутами src и alt
- 📰 Всех заголовков (h1, h2, h3)
- 📊 Таблиц с данными
- 📝 Списков (ul, ol)

6. Обработка ошибок парсинга
- ⚠️ Обрабатывать некорректный HTML
- 📝 Логировать предупреждения при парсинге
- 🔄 Возвращать частичные результаты при ошибках

7. Демонстрация
Создать скрипт, который:
- 📥 Загружает и парсит несколько страниц
- 🔗 Извлекает заголовки и ссылки
- 💾 Показывает итог в структурированном и понятном виде
- 📊 Показывает статистику (количество ссылок, длина текста)

Пример вывода:
```python
{
    "url": "https://example.com",
    "title": "Example Domain",
    "text_length": 1234,
    "links_count": 5,
    "links": ["https://example.com/page1", ...],
    "images_count": 2
}
```

8. Тестирование
- ✅ Протестировать парсинг валидного HTML
- ⚠️ Протестировать обработку битого HTML
- 🔗 Протестировать извлечение ссылок
- 🔄 Проверить конвертацию относительных URL

🎉 Критерии успеха
- ✅ HTML корректно парсится
- 📊 Данные извлекаются правильно
- 🔄 Относительные ссылки конвертируются в абсолютные
- 🛡️ Ошибки парсинга не ломают программу
