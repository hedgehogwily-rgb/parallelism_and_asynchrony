День 1 — Базовый асинхронный HTTP-клиент

🎯 Цель дня
Создать базовый асинхронный HTTP-клиент для загрузки веб-страниц с использованием aiohttp.

📋 Требования

1. Установка зависимостей
Установить необходимые библиотеки:
```bash
pip install aiohttp aiofiles
```

2. Базовый класс AsyncCrawler
Создать класс AsyncCrawler с методами:
- __init__(self, max_concurrent: int = 10) — инициализация с ограничением конкурентности
- async def fetch_url(self, url: str) -> str — загрузка одной страницы
- async def fetch_urls(self, urls: list[str]) -> dict[str, str] — параллельная загрузка списка URL
- async def close(self) — закрытие сессии

3. Использование aiohttp.ClientSession
- 🌐 Создать сессию с таймаутами (connect, read)
- 🔗 Использовать connection pooling
- ✅ Правильно закрывать сессию после использования

4. Обработка ошибок
Обрабатывать следующие исключения:
- ❌ aiohttp.ClientError — сетевые ошибки
- ⏰ asyncio.TimeoutError — таймауты
- 🚫 aiohttp.ClientResponseError — HTTP ошибки (404, 500 и т.д.)

5. Базовое логирование
Логировать:
- ▶️ начало загрузки URL
- ✅ успешное завершение
- ⚠️ ошибки с указанием URL и типа ошибки

6. Демонстрация
Создать скрипт, который:
- 📥 загружает 5-10 различных URL параллельно
- 📊 выводит статус каждого запроса
- ⏱️ измеряет общее время выполнения
- 🚀 показывает преимущество параллельной загрузки

Пример использования:
```python
async def main():
    crawler = AsyncCrawler(max_concurrent=5)
    urls = [
        "https://example.com",
        "https://httpbin.org/delay/1",
        "https://httpbin.org/delay/2",
    ]
    results = await crawler.fetch_urls(urls)
    await crawler.close()
    print(f"Загружено {len(results)} страниц")

asyncio.run(main())
```

7. Тестирование
- ✅ Протестировать загрузку валидных URL
- ❌ Протестировать обработку несуществующих URL
- ⏰ Протестировать таймауты
- ⚡️ Сравнить время последовательной и параллельной загрузки

🎉 Критерии успеха
- ✅ Класс корректно загружает страницы асинхронно
- 🛡️ Ошибки обрабатываются без падения программы
- 📝 Логирование работает
- ⚡️ Параллельная загрузка быстрее последовательной
