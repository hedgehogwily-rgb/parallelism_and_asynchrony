День 3 — Управление конкурентностью и очередями

🎯 Цель дня
Реализовать систему управления конкурентностью, очередей задач и контроля загрузки.

📋 Требования

1. Класс CrawlerQueue
Создать класс для управления очередью URL:
- add_url(self, url: str, priority: int = 0) — добавление URL с приоритетом
- async def get_next(self) -> str | None — получение следующего URL
- mark_processed(self, url: str) — пометка URL как обработанного
- mark_failed(self, url: str, error: str) — пометка URL как неудачного
- get_stats(self) -> dict — статистика очереди

2. Класс SemaphoreManager
Реализовать управление семафорами:
- 🚦 Ограничение количества одновременных запросов к одному домену
- ⚡ Глобальное ограничение конкурентности
- 📊 Отслеживание активных задач

3. Расширенный AsyncCrawler
Добавить в AsyncCrawler:
- async def crawl(self, start_urls: list[str], max_pages: int = 100) — основной метод краулинга
- 📋 Использование очереди для управления URL
- ➕ Автоматическое добавление найденных ссылок в очередь
- 📏 Ограничение глубины обхода (max_depth)
- 🔍 Фильтрация уже посещённых URL

4. Управление состоянием
Реализовать:
- visited_urls: set — множество посещённых URL
- failed_urls: dict — словарь неудачных URL с ошибками
- processed_urls: dict — словарь обработанных URL с результатами

5. Контроль глубины обхода
- 📏 Параметр max_depth для ограничения глубины
- 📊 Отслеживание текущей глубины для каждого URL
- ⏭️ Пропуск ссылок, превышающих максимальную глубину

6. Фильтрация URL
Реализовать фильтры:
- 🏠 Только тот же домен (same_domain_only)
- ❌ Исключение определённых паттернов (exclude_patterns)
- ✅ Включение только определённых паттернов (include_patterns)

7. Прогресс и статистика
Выводить в реальном времени:
- 📄 Количество обработанных страниц
- ⏳ Количество в очереди
- ❌ Количество ошибок
- ⚡ Скорость обработки (страниц/сек)

8. Демонстрация
Создать скрипт, который:
- 📍 Начинает с одного или нескольких стартовых URL
- 🌐 Обходит сайт с ограничением глубины
- 📊 Показывает прогресс в реальном времени
- 💾 Сохраняет все найденные данные

Пример использования:
```python
async def main():
    crawler = AsyncCrawler(max_concurrent=10, max_depth=2)
    results = await crawler.crawl(
        start_urls=["https://example.com"],
        max_pages=50,
        same_domain_only=True
    )
    print(f"Обработано: {len(results)} страниц")

asyncio.run(main())
```
